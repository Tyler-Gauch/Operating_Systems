Assignment 1
=============

This program takes in two command line arguments, an input file and an output file.  It then reverses the input file byte by byte and writes the reversed file in the output file.

USAGE: ./assignment1 <inputfile> <outputfile> <traversetype>

<traversetype>: 0 || 1 || 2
<inputfile>: Existing File
<outputfile>: File to store the reversed file in
